Deck n'Drop is a react-native app using firebase for authentication and firestore as storage.

## Technology
- React-Native
- Redux
- REST API
- expo
- Firebase

## Getting Started
- `git clone <url>`
- `npm install`
- `cd ios && pod install`
- open xCode
- open deckndrop.xcworkspace
- run project
